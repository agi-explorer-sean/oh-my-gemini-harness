# parallel-exec

This feature implements stateless parallel execution parallels.

## File Sources

### Local Files

A unique Gemini capability for massive scale parallelization: -
`coordinator.ts`: Orchestrates parallel agents. - `eval.ts`: Evaluation logic
for parallel tasks. - `isolation.ts`: Environment isolation for parallel
agents. - `synthesizer.ts`: Logic for merging results from multiple agents. -
`storage.ts`: Persistence for parallel state. - `types.ts`: Parallel execution
types. - `index.ts`: Feature exports. - `*.test.ts`: Integration and unit tests
for the parallel feature.
